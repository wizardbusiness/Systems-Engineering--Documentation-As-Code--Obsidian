## About

##### *User Accounts* are a type of [Security Principle](https://learn.microsoft.com/en-us/windows-server/identity/ad-ds/manage/understand-security-principals) entity. Users can authenticate to a windows operating system using the sign-in process. Doing so grants the user an access token which allows the user to access information and run applications for which that user has privileges.
