
### On Debian Linux

#### *Check if Pip is installed*
```ad-bash
	pip python -m pip -v
```

#### 1. Update Debian
```ad-bash
collapse: true
	sudo apt update	
```

#### 2. Install pip
##### **A) Python3** (Preferred)
```ad-bash
collapse: true
	sudo apt install python3-pip
```

##### **B) Python**
```ad-bash
collapse: true
	sudo apt install python-pip
```
