### Ansible

_**An open-source automation tool that manages and configures multiple computers simultaneously without requiring agent software on the target machines.**_

### Control Node

_**The computer where Ansible is installed and from which [[Shared Glossary#Automation Command|automation commands]] and [[_Ansible Glossary#Playbook|playbooks]] are executed to manage other systems.**_

### Playbook

_**A YAML file containing a series of automated tasks and configurations that Ansible [[Shared Glossary#Execution|executes]]on target systems in a defined sequence.**_
