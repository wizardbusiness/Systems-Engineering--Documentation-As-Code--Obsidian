##### **` Application `**
  ***A software program designed to perform specific tasks for users, such as word processing, web browsing, or gaming***
****
##### **` Program `**

  ***A set of programmed instructions that tells a computer what tasks to perform.*** 

****
#### **` Service `**
  ***A program that runs in the background of an operating system, performing specific tasks or providing functionality to other programs without direct user interaction.*** 
****
