### Execution

_**The process of a computer running or carrying out the instructions in a program, line by line, to perform the intended tasks.**_
***
### Agent software

_**A program that is installed on a target computer to enable remote management or monitoring by another system.**_
***

### Automation Command

_**An instruction that tells a computer or software to automatically perform a specific task or series of tasks without requiring manual input each time.**_
***

### Automation Tool

_**Software that performs repetitive tasks or processes automatically without manual human intervention.**_
***

### Configure

_**To set up or adjust the settings and parameters of a computer system or software to work in a desired way.**_
***

### Data

_**Information stored in digital form that computers can process, including numbers, text, images, and other content.**_
***

### Instructions

_**Individual commands or steps written in code that tell a computer to perform specific operations.**_
***

### Operating System

_**The foundational software that manages computer hardware and provides services for application programs to run.**_
***

### Open-source

_**Software whose source code is publicly available for anyone to view, modify, and distribute freely.**_
***

### Software

_**Instructions and data that tell computer hardware what to do, including all programs, applications, and operating systems that run on a computer.**_

***

### Target Machine

_**A computer or device that is being managed, configured, or operated on by another system or administrator.**_

***

### YAML

_**A human-readable data format that uses indentation and simple syntax to structure configuration files and data, commonly used for settings, deployment scripts, and data exchange between systems (See [[YAML Example]])**_
***

### Computer Hardware

_**The physical components of a computer system, such as the processor, memory, storage drives, and input/output devices.**_
***

### Programs

_**Sets of organized instructions that tell a computer how to perform particular tasks or solve specific problems.**_
***

### Applications

_**Software programs designed to help users accomplish specific tasks, such as writing documents, browsing websites, or editing photos.**_
***

### Operating Systems

_**The fundamental software that manages computer hardware and provides services for other programs to run, such as Windows, macOS, or Linux.**_
